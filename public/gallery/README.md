# Gallery Folder

Drop your gallery images directly into this folder. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`.

The gallery page at `/gallery` will automatically detect every image in this folder at build time — no code changes needed.

**Naming tip:** Use descriptive filenames like `tempo-traveller-jaipur-2024.jpg` — they're auto-converted into readable alt text for accessibility and SEO.

**Recommended:** Keep images under 500 KB each. Use [squoosh.app](https://squoosh.app) to compress before uploading. Around 100 images is comfortable; more is fine.

Once you've added or removed images, redeploy the site (Vercel/Netlify will rebuild automatically on push).
